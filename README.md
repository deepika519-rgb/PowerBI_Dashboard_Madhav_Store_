Project Overview

This project presents an interactive Power BI dashboard built to analyze sales performance of Madhav Store. The dashboard provides actionable insights into revenue, 
customer behavior, product performance, and regional trends. It helps stakeholders make data-driven decisions by visualizing key business metrics in a clear and 
dynamic way.

Tools

1.Power BI
2.Data Cleaning & Transformation (Power Query)
3.DAX (Data Analysis Expressions)
4.Data Visualization

Dataset Description

The dataset contains sales-related information such as:
* Orders and transactions
* Customer details
* Product categories
* Sales amount and quantity
* Regional data

Key Dashboard Features

🔹 Sales Overview
Total Sales, Profit, and Quantity KPIs
Monthly and yearly sales trends
Dynamic filtering using slicers

🔹 Customer & Region Insights
Top customers by revenue
Sales distribution by region
Customer purchase patterns

🔹 Product Performance
Best-selling product categories
Product-wise revenue contribution
Quantity vs sales comparison

Key Insights

* Identified top-performing regions contributing maximum revenue
* Highlighted high-value customers for targeted marketing
* Discovered best-selling product categories
* Tracked seasonal sales trends
